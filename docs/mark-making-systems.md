# Mark-Making Systems — outsider_art_style

## Overview

This document describes the mark-making logic implemented in each shader and the specific outsider art aesthetic regimes they embody. These are not approximations of academic painting — they are distillations of the **urgency, repetition, and directness** of art made outside institutions.

---

## Core Mark Primitives

### 1. Crayon Grain (`crayonLayer`, `crayonStroke`)

**What it is:**  
Waxy parallel strokes with slight directional variation. The direction field is generated by fbm noise at low frequency — the hand naturally rotates the paper slightly as it works.

**Math:**
```glsl
float strokePos = fract(dot(uv, perp) * frequency);
float grain = noise(uv * 200.0) * 0.4 + 0.6;  // wax texture
float pressure = noise(uv * 8.0) * 0.5 + 0.5;  // pressure variation
```

**Outsider reference:**  
The crayon stroke is the most democratic mark. No brush to clean, no brush to ruin. Just hand to wax to paper. Finster used crayon on board. The marks are visible, parallel, and direct.

**Shader:** `09_crayon_field.frag.glsl` — pure crayon layer study

---

### 2. Mark Wobble (`wobble`, `wobbleSlow`)

**What it is:**  
Path displacement via noise — the hand trembles, the marker wavers. This is NOT imperfection. It is the **pulse of the maker** made visible in the line.

**Math:**
```glsl
vec2 wobble(vec2 uv, float amount) {
    float wx = noise(uv * 50.0 + vec2(1.7, 0.0)) - 0.5;
    float wy = noise(uv * 50.0 + vec2(0.0, 2.3)) - 0.5;
    return uv + vec2(wx, wy) * amount;
}
```

**u_mark_wobble range:** 0.0 (mechanical, cold) → 0.3 (trembling, alive)

**Outsider reference:**  
Madge Gill's automatic drawing. The spirit guide moved her hand. Wölfli drawing alone in his room. The wobble is not corrected because correction implies doubt, and these makers have no doubt.

---

### 3. Newspaper Ground (`newspaperGround`, `newspaper`)

**What it is:**  
Simulation of a newspaper broadsheet substrate — columns, text lines, headlines, photo boxes. The original material is legible beneath the paint.

**Math:**
```glsl
// Column structure (6 columns)
float cols = mod(uv.x * 7.0, 1.0);
float colRule = smoothstep(0.0, 0.02, cols) * (1.0 - smoothstep(0.96, 1.0, cols));
// Text line simulation
float lineY = mod(uv.y * lineSpacing, 1.0);
float textLine = step(0.3, lineY) * step(lineY, 0.85);
// Individual character (hash-based occupancy)
float charSim = step(0.3, hash(vec2(charX, charRow)));
```

**u_ground_show range:** 0.0 (paint covers entirely) → 1.0 (raw substrate, paint almost gone)

**Outsider reference:**  
Henry Darger collected newspapers and used them as collage elements in his paintings. The text columns showing through painted flowers is an accident that became intention.

### 4. Horror Vacui Fill (`recursiveFill`, `patternFill`)

**What it is:**  
Recursive subdivision of the picture plane until no empty space exists. Each level tiles a different motif (diamonds, stars, crosses, hatching). The terror of emptiness drives the filling.

**Math:**
```glsl
float fill0 = recursiveFill(wuv, 0.0);  // large diamonds
float fill1 = recursiveFill(wuv * 1.618 + 0.3, 1.0);  // medium stars
float fill2 = recursiveFill(wuv * 2.618 + 0.7, 2.0);  // small crosses
totalFill = max(fill0, max(fill1 * density, fill2 * density * 0.6));
```

**Golden ratio scaling** between levels creates self-similar density.

**u_horror_vacui range:** 0.0 (breathing room) → 1.0 (panic density)

**Outsider reference:**  
Emma Kunz spent entire nights filling graph paper with geometric networks. Wölfli left no margin unfilled. The anxiety of empty space is the engine of outsider art's density.

---

### 5. Nested Borders (`nestedBorder`)

**What it is:**  
Concentric frame lines at regular intervals. Each border is slightly different weight. The wobble varies per border level. The edge of the picture plane is as contested as its center.

**Math:**
```glsl
for (float i = 0.0; i < n; i++) {
    float off = i * gap;
    vec2 inner = (uv - off) / (1.0 - 2.0 * off);
    float bVal = borderWobble(inner, thickness, wobble);
    result = max(result, bVal);
}
```

**Outsider reference:**  
Wölfli's pages have 3–5 border levels. Finster painted the physical edges of his boards. The frame is not passive — it is a declaration.

---

### 6. Repetition Field (`fract(uv * density)`)

**What it is:**  
The most fundamental outsider art operation. Take one figure. Multiply it across the entire surface. Vary slightly per cell (rotation, scale, color, position jitter).

**Math:**
```glsl
vec2 cell = floor(uv * density);
vec2 local = fract(uv * density) - 0.5;  // centered in cell

// Per-cell variation
float cellH = hash(cell);
float cellScale = 0.85 + cellH * 0.25;  // size variation
float rotAngle = (hash(cell+0.5) - 0.5) * 0.35;  // tilt variation
```

**Outsider reference:**  
Darger's armies — the same girl-figure traced from coloring books, repeated hundreds of times. Bill Traylor's figures repeated in rows. The repetition is prayer, proof, accumulation.

---

### 7. Text Texture (`textTexture`, `handwrittenLine`)

**What it is:**  
Words as visual marks. Not for reading. The density of text creates a texture. Handwriting is simulated through variable letter width, baseline wobble, and multiple "letter type" templates.

**Math:**
```glsl
// Baseline wobble
float baselineWobble = sin(uv.x * 8.0 + seed) * wobble * lineH * 0.3;
// Letter width variation (not monospace — handwritten)
float letterW = charAvgW * (0.35 + cid * 0.50);
// Ink weight at stroke center
float strokeWeight = sin(charX / letterW * 3.14159) * 0.4 + 0.6;
```

**u_text_density range:** 0.0 (silent image) → 1.0 (image as dense text)

**Outsider reference:**  
Finster labeled everything. Wölfli's pages are equal parts image and text. The maker cannot remain silent. The image MUST speak.

---

### 8. Naive Figure (`naiveFigure`, `naiveFace`, `angelFigure`)

**What it is:**  
A body rendered by someone who knows what a body IS but not exactly how it works. Proportions slightly wrong but emotionally correct. The head is always too large — the most important part. The arms are exactly as long as needed.

**Construction:**
- Head: sphere/ellipse, slightly oversized
- Body: rectangle or trapezoid (the dress/robe)
- Arms: horizontal, drooped slightly (gravity acts on kindness)
- Legs: parallel, slightly apart
- Wings (angel): triangular extensions, the most important element

**Outline:** Thick, consistent, slightly wobbly. Not tapered. The line knows what it is.

**Outsider reference:**  
Darger's Vivian Girls — traced from coloring books, but owned. Bill Traylor's stick-figure-adjacent figures. Martin Ramirez's horses. The "incorrect" anatomy is not failure — it is *emphasis*.

## Aesthetic Regimes

### HORROR_VACUI
Every pixel claimed. Multiple fill systems layered. u_horror_vacui at 0.9–1.0.

### VISIONARY_MAP
Geometric cosmology as diagram. Voronoi territories. Cardinal structure. The personal universe made spatial.

### REPEATED_FIGURE
`fract(uv * density)` as prayer beads. One figure, many instances, slight variations as evidence of the maker's attention to each.

### FOUND_GROUND
u_ground_show at 0.5–1.0. The substrate is the partner. Paint negotiates with the existing world.

### TEXT_IMAGE
u_text_density at 0.7–1.0. The silence of the image is broken by necessity. Every name must be spoken. Every thing must be named.

## Notes on Authenticity

These shaders do not appropriate specific works. They distill **the logic** of how outsider art is made:

1. **The compulsion is real.** The horror vacui fill logic genuinely panics when confronted with empty space.
2. **The wobble is alive.** `u_mark_wobble > 0` changes the character of every stroke.
3. **The palette is non-negotiable.** These are not chosen for beauty. They are chosen because they were what was available, or because they were the right color and the maker knew it without knowing why.
4. **Repetition is not laziness.** The `fract(uv * density)` operation is genuinely the same cognitive act as Wölfli filling margins with faces — the hand that cannot stop.

The wobble is not imperfection. It is the pulse of the maker's hand.
